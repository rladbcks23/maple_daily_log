# 사용 API 목록

메이플 숙제알리미에서 사용할 Nexon OpenAPI를 정리한다.

## 1. 계정/캐릭터 선택 API

넥슨 로그인을 활용하고, 해당 넥슨 계정에 있는 캐릭터 중 일부만 알림 대상으로 선택할 수 있게 한다.
앱에서는 사용자가 캐릭터를 구분할 수 있도록 기본 정보를 함께 보여준다.

### 캐릭터 목록 조회

- API: `/maplestory/v1/character/list`
- 필요 값:
  - API key
- 사용 목적:
  - 넥슨 계정에 있는 모든 캐릭터 목록 호출
  - 알림 대상으로 사용할 캐릭터 선택 화면 구성

### OCID 조회

- API: `/maplestory/v1/id`
- 필요 값:
  - API key
  - 캐릭터명
- 사용 목적:
  - 캐릭터 스케줄러 조회에 필요한 `ocid` 호출
  - 선택한 캐릭터의 일일/주간/보스 콘텐츠 조회 준비

### 캐릭터 기본 정보 조회

- API: `/maplestory/v1/character/basic`
- 필요 값:
  - API key
  - `ocid`
  - 날짜
- 날짜:
  - 미작성 시 오늘 기준으로 조회한다.
- 사용 목적:
  - 앱에서 선택 가능한 캐릭터를 구분할 수 있게 기본 정보를 제공
- 사용 정보:
  - 캐릭터 사진
  - 월드명
  - 캐릭터명
  - 직업
  - 레벨

## 2. 스케줄러 API

### 캐릭터 진행 상태 조회

- API: `/maplestory/v1/scheduler/character-state`
- 필요 값:
  - API key
  - `ocid`
- 내용:
  - 조회 기준일
  - 캐릭터명
  - 월드명
  - 레벨
  - 직업
  - 일일 콘텐츠 정보
  - 주간 콘텐츠 정보
  - 보스 콘텐츠 정보
  - 주간 보스 처치 완료 횟수
  - 주간 보스 처치 제한 횟수

### 확인된 제약

- 날짜 값을 넣지 않으면 오늘 기준 데이터를 조회한다.
- 오늘 해당 캐릭터로 접속하지 않은 상태에서는 `daily_contents`가 빈 배열로 내려올 수 있다.
- 접속 후 몇 분이 지나 다시 조회하면 인게임 스케줄러 기준 `daily_contents`가 채워진다.
- `weekly_contents`도 같은 방식으로 접속 이후에 채워질 가능성이 있다.
- 따라서 빈 배열을 곧바로 "할 일이 없음" 또는 "모두 완료"로 판단하지 않는다.

### 알림 판단 기준

- 접속 전 조회에서 `daily_contents`가 비어 있으면 "오늘 접속 전이라 스케줄러 정보를 확인할 수 없음"으로 본다.
- 미접속 상태에서는 일일 콘텐츠 미완료 목록 대신 접속 필요 알림을 우선한다.
- 메이플 런처 종료 후에는 다시 스케줄러 API를 조회한다.
- 런처 종료 후에도 콘텐츠 목록이 비어 있으면 완료 여부를 확정하지 않고 확인 불가 상태로 처리한다.

### 공통 세부 내용

일일 콘텐츠 정보, 주간 콘텐츠 정보, 보스 콘텐츠 정보는 세부 내용을 가진다.

- 콘텐츠명 또는 보스명
- 인게임 등록 여부
- 현재 완료 횟수 또는 점수
- 최대 완료 가능 횟수 또는 점수

### 일일 콘텐츠 정보

- 타입: `contents`, `quest`
- 퀘스트인 경우 진행 상태:
  - `0`: 기타
  - `1`: 진행 중
  - `2`: 완료

### 주간 콘텐츠 정보

- 퀘스트인 경우 진행 상태:
  - `0`: 기타
  - `1`: 진행 중
  - `2`: 완료

### 보스 콘텐츠 정보

- 완료 여부: `true`, `false`
- 리스트 순서
- 보스 초기화 주기
- 보스 난이도

## 3. 공지 정보 조회 API

공지 정보는 목록 조회와 상세 조회로 나누어서 관리한다.

### 목록 조회

- 공지사항 목록 조회: `/maplestory/v1/notice`
- 진행 중 이벤트 목록 조회: `/maplestory/v1/notice-event`
- 캐시샵 공지 목록 조회: `/maplestory/v1/notice-cashshop`
- 업데이트 목록 조회: `/maplestory/v1/notice-update`

### 상세 조회

- 공지사항 상세 조회: `/maplestory/v1/notice/detail`
- 업데이트 상세 조회: `/maplestory/v1/notice-update/detail`
- 진행 중 이벤트 상세 조회: `/maplestory/v1/notice-event/detail`
- 캐시샵 공지 상세 조회: `/maplestory/v1/notice-cashshop/detail`

### 목록 공통 내용

- 제목
- 링크
- 식별자
- 등록일

### 상세 공통 내용

- 제목
- 링크
- 본문
- 등록일

### 예외 내용

진행 중 이벤트 상세 목록/조회:

- 이벤트 시작일
- 이벤트 종료일

캐시샵 공지 상세 목록/조회:

- 판매 시작일
- 판매 종료일
- 상시 판매 여부
