# 사용 API 목록

메이플 숙제알리미에서 사용할 Nexon OpenAPI를 정리한다.

## 1. 스케줄러 API

### 캐릭터 진행 상태 조회

- API: `/maplestory/v1/scheduler/character-state`
- 내용:
  - 조회 기준일
  - 캐릭터명
  - 월드명
  - 레벨
  - 직업
  - 일일 콘텐츠 정보
  - 주간 콘텐츠 정보
  - 보스 콘텐츠 정보
  - 주간 보스 처치 완료 횟수
  - 주간 보스 처치 제한 횟수

### 공통 세부 내용

일일 콘텐츠 정보, 주간 콘텐츠 정보, 보스 콘텐츠 정보는 세부 내용을 가진다.

- 콘텐츠명 또는 보스명
- 인게임 등록 여부
- 현재 완료 횟수 또는 점수
- 최대 완료 가능 횟수 또는 점수

### 일일 콘텐츠 정보

- 타입: `contents`, `quest`
- 퀘스트인 경우 진행 상태:
  - `0`: 기타
  - `1`: 진행 중
  - `2`: 완료

### 주간 콘텐츠 정보

- 퀘스트인 경우 진행 상태:
  - `0`: 기타
  - `1`: 진행 중
  - `2`: 완료

### 보스 콘텐츠 정보

- 완료 여부: `true`, `false`
- 리스트 순서
- 보스 초기화 주기
- 보스 난이도

## 2. 공지 정보 조회 API

공지 정보는 목록 조회와 상세 조회로 나누어서 관리한다.

### 목록 조회

- 공지사항 목록 조회: `/maplestory/v1/notice`
- 진행 중 이벤트 목록 조회: `/maplestory/v1/notice-event`
- 캐시샵 공지 목록 조회: `/maplestory/v1/notice-cashshop`
- 업데이트 목록 조회: `/maplestory/v1/notice-update`

### 상세 조회

- 공지사항 상세 조회: `/maplestory/v1/notice/detail`
- 업데이트 상세 조회: `/maplestory/v1/notice-update/detail`
- 진행 중 이벤트 상세 조회: `/maplestory/v1/notice-event/detail`
- 캐시샵 공지 상세 조회: `/maplestory/v1/notice-cashshop/detail`

### 목록 공통 내용

- 제목
- 링크
- 식별자
- 등록일

### 상세 공통 내용

- 제목
- 링크
- 본문
- 등록일

### 예외 내용

진행 중 이벤트 상세 목록/조회:

- 이벤트 시작일
- 이벤트 종료일

캐시샵 공지 상세 목록/조회:

- 판매 시작일
- 판매 종료일
- 상시 판매 여부
