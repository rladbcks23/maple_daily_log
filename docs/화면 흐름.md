# 화면 흐름

클로드 디자인으로 만든 데스크톱 UI 구상을 기준으로 화면 흐름을 정리한다.
디자인 참고 파일은 `design/claude-ui/메이플 숙제알리미.dc.html`에 둔다.

## 1. 전체 레이아웃

- 좌측 사이드바와 우측 메인 영역으로 구성한다.
- 사이드바 상단에는 앱 이름과 현재 선택 캐릭터 요약을 보여준다.
- 사이드바 메뉴는 캐릭터 선택, 스케줄러, 이벤트, 공지사항, 이번주 선데이로 구성한다.
- 메인 상단에는 현재 페이지 제목과 놓친 알림 미리보기 버튼을 둔다.

## 2. 캐릭터 선택

- 넥슨 계정 캐릭터 목록을 카드 형태로 보여준다.
- 각 캐릭터 카드에는 캐릭터 이미지, 캐릭터명, 직업, 레벨을 표시한다.
- 선택된 캐릭터는 선택됨 배지를 표시한다.
- 캐릭터 추가 버튼을 통해 알림 대상 캐릭터를 추가할 수 있게 한다.
- 선택한 캐릭터는 `selectedCharacters`에 저장한다.

## 3. 스케줄러

- 일일 콘텐츠, 주간 콘텐츠, 보스 콘텐츠를 한 화면에서 확인한다.
- 일일 콘텐츠는 퀘스트와 콘텐츠로 나누어 보여준다.
- 주간 콘텐츠는 퀘스트와 콘텐츠로 나누어 보여준다.
- 보스 콘텐츠는 일간, 주간, 월간으로 나누어 보여준다.
- 완료된 항목과 미완료 항목은 배경색으로 구분한다.
- 진행 횟수나 점수가 있는 항목은 현재 값과 최대 값을 함께 표시한다.

## 4. 이벤트

- 진행 중인 이벤트를 카드 형태로 보여준다.
- 각 이벤트에는 썸네일, 제목, 기간을 표시한다.
- 새 이벤트, 진행 중, 종료 임박 같은 상태 표시는 추후 필요하면 추가한다.

## 5. 공지사항

- 공지사항 목록을 리스트 형태로 보여준다.
- 전체, 공지, 점검, GM소식 같은 필터 탭을 둔다.
- 각 공지에는 분류, 제목, 등록일을 표시한다.

## 6. 이번주 선데이

- 이번주 선데이 메이플 이미지를 보여준다.
- 이미지는 공지/이벤트 API에서 가져온 정보 또는 별도 이미지 URL을 사용한다.

## 7. 알림 팝업

- 놓친 숙제가 있으면 화면 오른쪽 아래에 알림 팝업을 띄운다.
- 알림에는 선택 캐릭터명과 놓친 항목 요약을 표시한다.
- 버튼은 닫기와 지금 확인하기로 구성한다.
- 지금 확인하기를 누르면 스케줄러 화면으로 이동한다.

## 8. 로그아웃 확인

- 로그아웃 버튼을 누르면 확인 모달을 띄운다.
- 로그아웃 시 넥슨 계정 세션이 종료되고 알림도 함께 꺼진다는 안내를 보여준다.
